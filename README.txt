Данный репозиторий содержит дерикторию "ИБ Хакатон финал" с файлом информационной базы для десктопного/серверного клиента

APK файлы для мобильного клиента (см. инструкцию для тестирования мобильного клиента) доступны в папке "Мобильный клиент", расположенной в виде дочерней директории на яндекс диске по ссылке https://disk.yandex.ru/d/eLpR9yYPgKCyPQ

======================================================================
Инструкция для запуска десктопного клиента
======================================================================
1. Имортировать файл информационной базы из директории "ИБ Хакатон финал" в предварительно созданную локальную ИБ 1С: Предприятие 8.3.25.1394
2. Перезапустить конфигуратор
3. Войти под данными:
- Логин: "Администратор"
- Пароль: "Администратор"
Примечание: в случае, если конфигуратор запросит подключение к хранилищу конфигураций, рекомендуется нажать на кнопку "Отменить"

======================================================================
Инструкция для тестирования мобильного клиента
======================================================================
1. Поставить файл ИБ на 1С Предприятие на Сервере
2. Опубликовать веб-сервер 1С Предприятия с ИБ с шага 1
3. Установить один из мобильных клиентов на основе файлов из соответствующей облачной папки
4. При запуске мобильного клиента указать IP адрес опубликованного сервера с ИБ
